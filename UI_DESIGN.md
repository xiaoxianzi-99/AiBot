## 颜色方案 / Color Scheme

### 主题色
- **主色调（Primary）**: 渐变紫色 (#667eea → #764ba2)
- **背景色（Background）**: 浅灰色 (#f5f5f5)
- **用户消息（User Message）**: 渐变紫色气泡
- **AI消息（AI Message）**: 白色气泡带阴影

### 详细样式

#### 用户消息气泡
```
┌──────────────────────────┐
│ 你 (粗体白色)             │
│ 用户的消息内容...         │
│ (白色文字)                │
└──────────────────────────┘
背景：渐变紫色 (右对齐)
阴影：轻微投影
圆角：15px
```

#### AI消息气泡
```
┌──────────────────────────┐
│ AI Bot (粗体深色)         │
│ AI的回复内容...           │
│ (深色文字)                │
└──────────────────────────┘
背景：白色 (左对齐)
边框：浅灰色
阴影：轻微投影
圆角：15px
```

## 交互效果 / Interactive Effects

### 发送按钮
- **正常状态**: 渐变紫色背景，白色文字
- **悬停状态**: 颜色加深，轻微放大 (scale: 1.05)
- **按下状态**: 颜色更深，轻微缩小 (scale: 0.98)
- **禁用状态**: 灰色背景，透明度降低

### 输入框
- **正常状态**: 浅灰背景，灰色边框
- **聚焦状态**: 紫色边框，边框加粗

### 滚动效果
- **自动滚动**: 新消息出现时自动滚到底部
- **平滑滚动**: 流畅的滚动动画

## 字体设置 / Typography

- **系统字体**: Microsoft YaHei (中文), Segoe UI (英文)
- **标题字体大小**: 24px (粗体)
- **副标题字体大小**: 14px
- **消息内容字体大小**: 14px
- **发送者名称字体大小**: 12px (粗体)

## 响应式设计 / Responsive Design

- **最小宽度**: 600px
- **最小高度**: 400px
- **默认大小**: 800px × 600px
- **最大消息气泡宽度**: 500px

## 动画效果 / Animations

1. **消息出现动画**: 新消息从底部淡入
2. **按钮交互动画**: 悬停和点击时的缩放效果
3. **输入框聚焦动画**: 边框颜色平滑过渡

## 特殊状态显示 / Special States

### 加载状态
- 发送按钮禁用
- 显示处理中的视觉反馈

### 错误状态
- 系统消息以不同样式显示错误信息
- 保持界面友好和可用

## 用户体验特性 / UX Features

1. ✅ **快捷键支持**: Enter 发送，Shift+Enter 换行
2. ✅ **自动焦点**: 发送后焦点返回输入框
3. ✅ **防重复发送**: 处理期间禁用发送按钮
4. ✅ **空消息验证**: 不允许发送空白消息
5. ✅ **消息自动换行**: 长消息自动换行显示
6. ✅ **滚动条美化**: 自定义滚动条样式

## 可访问性 / Accessibility

- 清晰的视觉层次
- 足够的颜色对比度
- 合理的字体大小
- 键盘操作支持

## 国际化支持 / i18n Support

当前界面支持：
- ✅ 中文界面
- ✅ 中英文混合内容显示

## 浏览器兼容性说明 / Compatibility Note

注意：这是一个 JavaFX 桌面应用，不是 Web 应用：
- ✅ Windows 10/11
- ✅ macOS 10.14+
- ✅ Linux (Ubuntu 18.04+)
- ⚠️ 需要 Java 17 或更高版本
- ⚠️ 需要 JavaFX 运行时环境
